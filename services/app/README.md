# Web Application Service

Flask Web 应用服务，提供数据分析仪表板和 REST API。

## 文件说明

- `app.py` - Flask 应用主文件
- `Dockerfile` - Docker 镜像构建文件
- `requirements.txt` - Python 依赖
- `templates/` - HTML 模板目录
- `static/` - 静态资源目录

## 功能

- 数据分析仪表板（http://localhost:3000）
- RESTful API 端点
- 实时图表和数据可视化

## 端口

- 3000 - Web 界面和 API
